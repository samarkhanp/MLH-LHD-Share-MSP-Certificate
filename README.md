# MLH LHD Share MSP Certificate

PDF Certificate generator for event attendees

  - Download certificate using email
  - Email Verification


### Usage
  - Import CSV to MySQL DB
  - Verify MySQL table column names
  - Add Certificate template as images
  - Edit 'generate.php'
